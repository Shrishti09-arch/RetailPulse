## 📌 Project Overview

RetailPulse is an AI-powered retail analytics platform that enables retailers to analyze sales performance, forecast demand, optimize inventory, monitor store performance, and generate AI-driven business insights through an interactive dashboard.

The project combines Machine Learning, Data Analytics, Forecasting, and Business Intelligence into one professional dashboard built using Streamlit.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/RetailPulse.git
```

Move into project

```bash
cd RetailPulse
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
cd dashboard

streamlit run login.py
```
